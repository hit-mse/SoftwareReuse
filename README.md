# SoftwareReuse - Stöck Mäster

## Requirements
* MongoDB 
* NPM

## Installation 
1. run "npm install" from the root directory.

## Start
1. Start mongoDB.
2. Run "npm start" in root directory to start the application.
3. Open http://localhost:4200 in your browser. (Recommended to use Chrome.)
